# Contributing

## Setup

```bash
git clone https://github.com/okeidar/super-worker.git
cd super-worker
pip install -e ".[dev]"
```

Prerequisites: Python 3.11+, tmux, and [Claude Code CLI](https://docs.anthropic.com/en/docs/claude-code).
